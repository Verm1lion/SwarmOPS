# Icon Dosyaları

Bu klasöre aşağıdaki icon dosyalarını eklemeniz gerekiyor:

- `icon.png` - 512x512 PNG (genel kullanım)
- `icon.ico` - Windows için ICO dosyası (256x256 veya daha büyük)
- `icon.icns` - Mac için ICNS dosyası

## Icon Oluşturma

1. **Online araçlar kullanarak:**
   - https://convertio.co/png-ico/ (PNG'den ICO'ya)
   - https://cloudconvert.com/png-to-icns (PNG'den ICNS'e)

2. **Manuel olarak:**
   - Windows: 256x256 veya 512x512 PNG'den ICO'ya dönüştürün
   - Mac: 512x512 PNG'den ICNS'e dönüştürün

3. **Geçici çözüm:**
   - Electron varsayılan icon'u kullanacaktır
   - İleride kendi icon'unuzu ekleyebilirsiniz
